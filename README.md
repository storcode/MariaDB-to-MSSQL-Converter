# MariaDB-to-MSSQL-Converter
**Оптимизированный конвертер SQL дампов из MariaDB/MySQL в Microsoft SQL Server**

## 📋 О проекте

##### Профессиональный инструмент для миграции данных из MariaDB/MySQL в Microsoft SQL Server. Поддерживает обработку больших дампов, автоматическое определение кодировок, переименование дублирующихся таблиц и интеллектуальную конвертацию типов данных.

🚀 Основные возможности

	Автоматическое определение кодировки SQL-дампа

	Парсинг CREATE TABLE и INSERT выражений

	Конвертация типов данных MariaDB → MSSQL

	Обработка дублирующихся имен таблиц

	Построчная вставка данных с обработкой ошибок

	Поддержка batch-обработки больших объемов данных

	Экранирование зарезервированных слов SQL Server

	Детальное логирование всех операций
